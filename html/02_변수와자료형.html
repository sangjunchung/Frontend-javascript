변수(Variable)

        * 메모리에 값을 계속 바꿔가면서 저장할 수 있는 공간, 변하는 수
        * 변수에 이름을 붙이고 (변수명) 원하는 값을 넣어주면
          변수명이 필요한 곳에서 변수명을 호출해서 지정된 값을 사용할 수 있음

        
변수명 이름 작명 규칙

        - 의미 있는 이름 작성(a,b,c 이런식으로 작성X)
        - 대, 소문자 작성
        - 특수문자는 _(언더바) 만 사용 가능(단, 첫 글자는 제외)
        - 중간에 대문자를 작성해주는 카멜표기법 작성

        특수 문자 중 $ 가 있음
        $는 혼자 사용할 수 있고,
        $숫자 사용할 수 있고,
        $변수명 맨 앞에 사용할 수 있음
        -> 추후 jQuery 표기 문법으로 사용할 문자

Java Script 변수의 종류와 특징

        var : 변수, 변수명 중복 O (덮어쓰기 가능)
            -> 변수명 재사용 가능, 변수명 안에 있는 내용 변경 가능
        let : 변수, 변수명 중복 X
            -> 변수명 재사용 불가능, 변수명 안에 있는 내용 변경 가능
        const : 상수, 변수명 중복 X
            -> 변수명 재사용 불가능, 변수명 안에 있는 내용 변경 불가능

자바스크립트의 자료형

        - 문자열을 String 사용 대신 "" 표현 후 작성
          const 문자열 = "hihi";
        - 정수를 표현할 때는 int 대신 숫자 작성
          const 정수 = 1;
        - 논리값(boolean)은 true false 작성
          const 논리값 = true;

        - null -> 참조하는 값이 없음 * 주로 오타에서 발생
