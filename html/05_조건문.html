조건문

        프로그래밍 코드가 해석되는 순서를 제어하는 제어문의 한 종류로
        작성된 조건식에 따라 수행되는 코드를 변경하는 구문

if 문

        
            if(조건식){
                조건식이 true일 경우 실행할 코드;
            }
        
        * 조건식 : 결과가 true, false 인 식

if - else 문

        
            if(조건식){
                조건식이 true 인 경우 수행할 코드;
            } else {
                조건식이 false 인 경우 수행할 코드;
            }
        
        * else 는 조건식을 작성하는 () 없음
0~100 사이에 랜덤 숫자를 발생시켜서 홀/짝 판별

if - else if - else 문

        
          if (조건식1) {
            조건식1 이 true 인 경우 발생;
          } else if (조건식2) {
            조건식2 가 true 인 경우 발생;
          } else {
            조건식1, 2 가 모두 false 인 경우 발생;
          }
어린이, 청소년, 성인 구분하기
        나이를 입력 받아 0 ~ 13 : 어린이, 14 ~ 19 청소년, 20 ~ 120 : 성인
