Javascript 개념 정리

1. var, let, const 차이점

        var : 재선언, 재할당(값 변경) 가능
            ex) var 변수 = 0;
                var 변수 = 1; // 재선언
                    변수 = 2; // 재할당

        let : 재선언 불가, 재할당 가능
            ex) let 변수 = 0;
                변수 = 1;

        const : 재선언, 재할당 불가
            ex) const 변수 = 0;

        재선언과 재할당이랑?
        재선언 : 기존에 할당되어 있는 변수를 다시 선언하여 새로운 정의를 내리는 것
        재할당 : 기존에 있던 변수의 값을 재할당하여 새로운 값을 저장하는 것
자료형 재선언 재할당

let X O

3. Javascript 객체에 대해 작성하시오.

객체는 자바스크립트에서 데이터를 구조화하여 저장하는데 사용하는 기본 단위

키(Key)-값(Value) 쌍으로 이루어져 있음

객체는 {} 중괄호를 사용해서 정의

키는 문자열로, 값은 어떠한 데이터 타입도 가능

      const 사람객체 = {
        '이름':'홍길동', 
        '나이':30, 
        '인사말': function(){
            alert('환영합니다.');
        }
      }

      사람 객체는 이름과 나이라는 속성을 가짐, 인사말이라는 메서드를 가지고 있음
      사람 객체
        속성 : 이름, 나이
        메서드 : 인사말

5. 백엔드 / 프론트엔드 / 풀스택 개발자의 차이점에 대해 작성하시오.

        백엔드
            서버, 데이터베이스, 데이터베이스와 프론트엔드 연결고리
